## 开发
1. 下载代码.
```
git clone https://github.com/willjw3/react-electron.git <your-project-name>
```
2. 安装yarn.
```
npm install -g yarn
```
3. 下载依赖.
```
cd <your-project-name> && npm install
```
4. 运行
```
npm run start
```
## 编译
1. 源码编译
```
npm run build
```
